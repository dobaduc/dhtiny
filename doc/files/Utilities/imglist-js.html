Introduction

This module support loading image before using to improve application speed.  Once this module is loaded, it can be called through $dhtiny.imgList

Properties

items

Set of all loaded images

defaultExt

Default image extension: gif.  In case of loading an image with unspecified extension, defaultExt will be used.

add

add: function(_id,
_src)

Load a new image with specified ID.

Parameters

_idid of the image _src = Source path of the image

setDefaultExt

setDefaultExt: function(ext)

Change default image extension

Parameters

extImage extension (Eg: jpg, png,...)

getItem

getItem: function(_src)

Get an image object, load the image if it does not exist in the loaded file list.

Return

image file path or null.

remove

remove: function(_id)

Remove an item from list

Parameter

_id_id of the image

setImage

src)

Set src property for an image after confirming about the image’s availbility

Parameter

imgThe image object

dispose

dispose: function()

Clean up all images
