Properties

contentArea

The content areaan instance of DHRichCanvas.  Default class name is “contentArea”

controller

The controller areaan instance of DHRichCanvas.  Default class name is “controller”

percentage

The percentage labelan instance of DHRichCanvas.  Default class name is “percentage”

slider

Seek baran instance of DHSlider.  Default class name is “slider”

sliderProps

Property set of animation seek bar.  See DHSlider for more details.

buttonProps

Array of controller buttons’ property set.  Refer <DHRichButton> to see more details about each button’s property set.  Following is an example

buttonProps = {
    // Normal DHRichButton properties
    position:"absolute",
    left: 40, top: 14, width: 30, cursor:"pointer",

    // List of images used for a controller button's statuses and events
    images:[
        { // Image 0
            src: "play.png",

            // Change background image to "Image 1" when click on the button with "Image 0" as current background
            clickIdx: 1,

            // Click event handler
            handler: function(animplayer, buttonIdx) {
                alert("Clicked on button "+buttonIdx+", info: "+ animplayer.buttons[buttonIdx].icon.src);
            }
        },
        { // Image 1
            src: "pause.png",

            // Change background image to "Image 0" when click on the button with "Image 1" as current background
            clickIdx: 0
        },
        { // Image 2
            src: "restart.png",

            // Change background image to "Image 0" when click on the button with "Image 2" as current background
            clickIdx: 1
        }
    ]
}
