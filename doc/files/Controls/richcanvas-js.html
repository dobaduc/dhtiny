Introduction

RichCanvas is a really simple but useful class that helps you to create and handle DOM objects easily.  This can be called the base class for most of DHTiny’s built-in controls.  Current version of RichCanvas just support HTML4 only, but HTML5 version will be presented soon with stronger HTML5 capability.

Example

var test = new DHRichCanvas(document.body, {
    position: "absolute",
     zIndex: 100,
     left: 520, top: 100, width: 100, height: 100,
     background: "red",
     onClick: function() {
        alert("Hello world!");
     }
 });
 test.addEv("onmouseover", function() {
    this.setProp("background","blue");
 });
 test.addEv("onmouseout", function() {
    this.setProp("background","red");
 });

setProp

setProp: function(p,
value)

Set value for a given property

Parameters

pProperty name.
valueProperty value.

NOTE

  • If a class method called “setSomeProperty” is defined, it will be called every time you call setProp(“someProperty”, value).
  • By default, you can set most of CSS properties by this method (width, height, fontSize, textAlign,...)
  • For the properties using “px” unit (Eg: top, left, width,...), we can bypass the “px” suffix and just specify the number value.
