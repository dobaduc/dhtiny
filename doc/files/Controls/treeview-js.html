setNodeData

setNodeData: function(node,
data)

Set data for an existing node at runtime

Parameters

nodeNode to be changed
dataData can be a string (node caption) or a property set

NOTE: node data format A single node’s data can be a string, a property set or a nested array of node data

// String
node_data1= "Node title"

// Property set
node_data2= {innerHTML: "Item1 - Simple tree view", iconType:"file", status: "expanded"}

// Nested node
[   //Layer 1
    {innerHTML: "Item3",iconType:"folder"},
    [ // Layer2
      {innerHTML: "Item31",iconType:"file"},
      [
          {innerHTML: "Item32",iconType:"folder"},
          ["A1","A2","A3"]
      ],
      {innerHTML: "Item33",iconType:"file"}
    ]
]
