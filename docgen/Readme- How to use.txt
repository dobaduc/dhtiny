DHTiny API document generation guide:
0. Make sure that active perl are installed on your computer.
   Download it from here: http://www.activestate.com/activeperl/downloads
1. Copy NEW modules ONLY from "[dhtiny root]/ctrl" and "[dhtiny root]/util" must be copied to "[dhtiny root]/docgen/dhtiny"
2. Follow NaturalDocs syntax guide (or refer to existing document source files in "[dhtiny root]/docgen/dhtiny" to write document for NEW modules.
   NaturalDocs guide: http://www.naturaldocs.org/documenting/walkthrough.html
3. Edit file "[dhtiny root]/docgen/temp/menu-mine" to make new modules appears on the document list.
4. Run "[dhtiny root]/docgen/GenDoc.bat"
5. Done!
